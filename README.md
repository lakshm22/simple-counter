# Simple Counter
A simple counter UI built using HTML, CSS and JavaScript.

---

### Purpose?
A mini project to learn the usage of CSS ID selectors, JS constants, functions and variable declaration.

----

### Tech Stack ⚒️
- `HTML5`
- `CSS`
- `JavaScript`
